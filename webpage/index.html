ASP Online
About
Want to try out some Answer Set Programming (ASP)? ASP Online is a web-based and interactive interface to the Potassco Answer Set Programming tool, oclingo.
Credits
The web UI & Backend Server were created by myself, Sean Policarpio. All ASP reasoning is performed on the server using the Potassco Answer Set Programming Tools by the University of Potsdam [http://sourceforge.net/projects/potassco/]
